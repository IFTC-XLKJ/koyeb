核心能力

不只是把 TS 转成 JS，还内置了一套可运行的运行时接口。

TS TypeScript 编译

将 TypeScript 源码编译为可执行的 JavaScript，命令行一键完成。

mod import 模块

支持 import 导入本地模块；暂不支持 npm 包。

net HTTP 服务器

可用 http.createServer 启动本地 HTTP 服务。

fs 文件系统

支持 fs 同步与异步读写，操作本地文件。

fetch 网络请求

内置 fetch，并支持 ReadableStream。

buf Buffer 与定时器

支持 Buffer、setTimeout / setInterval。

ui 交互对话框

支持 alert、confirm、prompt 基础交互。

os 跨平台

支持 Windows 与 Linux（含 Termux），依赖仅需 clang 与 curl。

// getting-started

快速上手

解压后用 Node 调用 CLI 即可编译。

1

安装依赖

确保系统已安装 Node.js，以及编译运行所需的系统依赖：

clang · curl · node
2

下载并解压

下载发行包，解压到任意目录。

mini-tsc.zip
3

编译 TypeScript

在解压目录下执行编译命令（将输入文件编译为输出文件）：

node dist/cli/index.js compile <input> -o <output>
4

示例

把 server.ts 编译为 server.js：

node dist/cli/index.js compile server.ts -o server.js
node server.js
