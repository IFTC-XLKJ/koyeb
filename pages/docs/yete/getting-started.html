开始使用

欢迎来到 Yete！这是一个简单的指南，帮助你快速上手。

Yete 是一个现代化的前端开发框架，旨在简化 Web 应用的开发过程。

安装

Yete CLI:

     Windows x64

下载完成后将可执行文件添加到系统 PATH 中。

创建项目

使用 Yete CLI 创建一个新的项目：

通过 chcp 65001 命令设置控制台编码为 UTF-8。

yete init my-project

项目目录：

my-project
├── dist/ # 静态资源输出目录
├── esbuild.config.js # esbuild构建配置文件
├── index.xml # 页面文件
├── index.js # 项目入口文件
├── package.json # 项目依赖
└── yete.config.json # Yete 配置文件

安装依赖

npm i

构建项目并运行开发服务器

进入项目目录并启动开发服务器：

cd my-project
yete build && yete serve

打开浏览器访问 http://localhost:3000，你应该能看到你的应用正在运行！
