开始使用

欢迎来到 Yete！这是一个简单的指南，帮助你快速上手。

Yete 是一个现代化的前端开发框架，旨在简化 Web 应用的开发过程。

使用 Dexie 库操作 IndexedDB，将资源文件缓存到浏览器中，能够减少加载时间和带宽消耗。

每次构建后，会重新下载一次资源文件，并缓存到浏览器中。

libs 目录下的文件不会视为资源文件，如需使用，请使用 命令添加。

官方社区

本框架将在每周末更新

安装

Yete CLI:

    Windows x64

    Linux arm64

    CLI源文件(其他操作系统请自行编译或直接使用 Txiki.js 运行)

下载完成后建议将可执行文件添加到系统 PATH 中。

创建项目

使用 Yete CLI 创建一个新的项目：

建议通过 chcp 65001 命令设置控制台编码为 UTF-8。

yete init my-project

项目目录：

my-project
├── dist/ # 静态资源输出目录
├── libs/ # 库目录
├── esbuild.config.js # esbuild构建配置文件
├── index.xml # 页面主文件
├── index.js # 项目入口文件
├── package.json # 项目依赖
└── yete.config.json # Yete 配置文件

安装依赖

npm i

构建项目并运行开发服务器

进入项目目录并启动开发服务器：

cd my-project
yete build && yete serve

打开浏览器访问 http://localhost:12345，你应该能看到你的应用正在运行！
